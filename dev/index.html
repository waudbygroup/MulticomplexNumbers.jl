MulticomplexNumbers.jl

A Julia package for representing multicomplex numbers and performing multicomplex algebra.

Authors

  • Chris Waudby, UCL School of Pharmacy, London (UK).

Citing

If you find this package useful, please cite this repository:

Waudby C. A. (2022). MulticomplexNumbers.jl. https://github.com/waudbygroup/MulticomplexNumbers.jl.

Installation

MulticomplexNumbers.jl is a registered package, and is installed by running

pkg> add("MulticomplexNumbers")
