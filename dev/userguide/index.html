User guide

For some simple examples, head over to the examples section. For a detailed guide, keep reading.

MulticomplexNumbers.jl is an implementation of multicomplex numbers. These numbers are a generalisation of complex numbers, recursively defined to contain multiple imaginary numbers, $i_1$, $i_2$ etc. A new type is defined, Multicomplex, which describes a multicomplex number of any order, and associated arithmetic operations.

The package is loaded as usual:

julia> using MulticomplexNumbers
